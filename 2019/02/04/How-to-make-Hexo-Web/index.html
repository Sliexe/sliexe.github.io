Hello Everybody. Welcome to web hexo. Hexo it’s so simple.

Berikut ini cara membuat web Hexo:

1. Membuat Akun dan Repository Github

Dengan membuat akun Github jika sudah membuat akun, berikutnya buat file Repository Github:
Profile > Your Repositories > New Repository > Isikan Nama Repository > Create Repository

LI3.png (New Repositories)

LI2.png (Create Repositories)

2. Install NodeJS and Hexo

Download software tersebut:
-NodeJS

Masuk ke Hexo :

  1. Masuk ke Command Prompt ketik perintah install Hexo

    Untuk menge-check sudah terinstall dan versi Hexo

3. How to Hexo web

Buat folder untuk asset file web hexo seperti gambar dibawah ini:
-

-Folder yang telah dibuat, masuk ke dalam Command Prompt untuk membuat Hexo web tersebut.

Untuk memulai Hexo web kalian, masukan command: hexo server
maka ada running localhost:4000 masukan command tersebut dari itulah web server Hexo kamu.

4.Install Hexo Theme and Deploy

Cari themes hexo di website Hexo Themes.

Sebelum lanjut kamu harus download text editor untuk editing script web hexo kamu. Saya pakai atom, untuk rekomendasi text editor pakailah atom atau notepad++ (Untuk mendowload Atom Click here)

Untuk Install Themes Hexo kalian lakukan lah sesuai instruksi tersebut contoh Install Themes Archer :

  1. Perintah instalasi ke command untuk penginstallan > npm i hexo-generator-json-content –save && npm i –save hexo-wordcount && git clone https://github.com/fi3ework/hexo-theme-archer.git themes/archer.

  2. Di dalam script folder web kamu ada _config.yml masuk ke text editor dan ubah bagian theme yang kalian pakai themes: Archer.

Deploy
1.hexo generate –watch
2.hexo clean
3.hexo deploy
Setelah mendeploy untuk yang pertama kali akan disuruh masuk ke akun github, dan folder website kalian akan masuk Repository Github dengan nama kalian buat masuk alamat website kamu username-kalian.github.io.

5. Make a Post

Masukan perintah di command hexo new post “judul kalian”, dengan begitu telah terbuat script di folder web kalian : source > _posts > judul-post-kalian.
Jika kalian ingin menaruh gambar ke postingan kalian, dengan pindahkan gambar ke dalam folder script post kalian. Kemudian pada script post tambahkan:

6. Enable on About

Di-tema saya mengaktifkan about, dengan membuat post baru:
hexo new page “about” > layout :about
Dan masuk ke _config.yml mengubah about menjadi true
about:
enable: false
image: /FolderGambar/Wallpaper-Kalian.png
